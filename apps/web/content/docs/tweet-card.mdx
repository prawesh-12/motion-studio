## Usage

A faithful X / Twitter post card that springs into frame. Use it for reaction videos, "look what people are saying" testimonial montages, social-proof reels, or as a hero element in product launches.

Toggle `theme` between `light` and `dark` to match the look you're going for. Engagement counts auto-format (`18432` → `18.4K`, `412000` → `412K`, `1500000` → `1.5M`). Drop any image URL into `avatarUrl` — leave it blank and it falls back to a clean gradient initial.
