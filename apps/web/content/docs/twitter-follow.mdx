## Usage

A profile card that mimics the X / Twitter follow flow. The card pops in, details stagger up, then a cursor flies onto the Follow button, clicks it, and the button morphs from filled "Follow" to outlined "Following" while the follower count ticks up by one.

Drop it into reaction videos, "go follow them" CTAs, social-proof reels, or anywhere you want to demonstrate a click. Toggle `theme` between `light` and `dark`. Any image URL works for `avatarUrl` — try `https://unavatar.io/twitter/<handle>` if you want avatars resolved straight from a handle.
