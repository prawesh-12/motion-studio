## Usage

Profile photo + slot-machine follower count + cascade of new-follower avatars. Each digit column in the count rolls vertically like a slot reel, and follower avatars pop in one by one as the count increments.

Avatars are resolved from `https://github.com/<username>.png` at render time, so any real GitHub username works — paste comma- or whitespace-separated handles in the "Recent follower usernames" field.

Brand-locked (uses GitHub's authentic palette in light or dark theme).
